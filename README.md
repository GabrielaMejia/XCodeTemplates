## Steps to install
1. Go to:
`
/Library/Developer/Xcode/Templates/
`

2. If folder `Custom Template` doesn&#39;t exists, create it with sentence:
`
mkdir Custom\ Templates
`
3. Paste template inside  `Custom Template` folder
4. Restart Xcode
